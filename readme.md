# Apocalipse [BR] - Pacote de Qualidade de Vida

Este é o pacote oficial de mods para o servidor **Apocalipse [BR]**.

**Discord:** [discord.gg/s8Rv3Kr8Y7](https://discord.gg/s8Rv3Kr8Y7)

## Sobre

Este repositório contém um conjunto de modificações desenvolvidas e mantidas pela equipe do servidor Apocalipse [BR]. Os mods incluídos foram cuidadosamente selecionados para melhorar a experiência de jogo dos jogadores do servidor.

## Instalação

### Via Steam Workshop
O pacote está disponível na Workshop do Steam. Inscreva-se na coleção oficial do servidor para obter automaticamente todas as atualizações.

### Instalação Manual
1. Faça o download do pacote completo
2. Extraia o conteúdo da pasta `Contents/mods/` para o diretório de mods do Project Zomboid
3. Ative os mods no menu de mods do jogo

## Compatibilidade

- **Versão Mínima:** Build 42.13
- **Versões Testadas:** Build 42.13 / Build 42.13
- **Multiplayer:** Compatível com servidor dedicado

## Suporte

Para suporte técnico ou reportar problemas:
- Entre em contato com a administração do servidor Apocalipse [BR]
- Abra uma issue neste repositório (somente para bugs técnicos)

## Aviso Importante

⚠️ Este pacote foi projetado especificamente para uso no servidor Apocalipse [BR]. Algumas funcionalidades podem não operar corretamente fora do ambiente do servidor.
